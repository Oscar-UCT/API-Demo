# API-Demo
Demo de una API en ASP NET

Pasos para testear
1. Clonar el repositorio
2. Ir a la carpeta del proyecto
3. Ejecutar "dotnet run" en la consola
4. Ir a localhost:puerto/swagger (herramienta para testear APIs)
